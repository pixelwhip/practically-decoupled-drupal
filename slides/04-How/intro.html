How to Decouple

- Various technical means of communicating between client and server - All are great for something - All of them have tradeoffs - We're going to work our way up the "coupling scale" - Some are highly coupled, the client and server need to "know" a lot about eachother - The last method, the client has no idea that it's talking to Drupal - Drupal has no idea if it's talking to a CLI, curl, or a browser
